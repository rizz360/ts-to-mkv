# Support

## Questions and Help

Use GitHub Discussions or open an issue for usage questions and troubleshooting.

When reporting issues, include:

- Your compose configuration (redacted)
- Relevant logs from /app/logs
- Input/output path layout
- ffmpeg/ffprobe version details

## Security Issues

For vulnerabilities, follow the process in SECURITY.md and avoid public issue reports.

## Feature Requests

Open a GitHub issue with the use case, expected behavior, and constraints.
